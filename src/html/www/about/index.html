About

About me

I'm Jake, a recent computer science graduate and aspiring software engineer. I've always been the type to take interest in things with flashing lights and bright colors, as long as I get to put it together myself. I began programming when I was 18—creating games in frameworks such as libGDX or XNA—and gained an interest in web design somewhere along the way. Nowadays, I can be found working on front-end UI, game engine architecture, or playing around with a Raspberry Pi. My language of choice at the moment is probably TypeScript, although I'm not afraid of diving into a C/C++ codebase.

Other interests of mine include various sports and hiking, which mostly balance out my other computer-based hobbies. I enjoy trying to learn foreign languages, and I dream of getting a pilot's license and traveling the world. Someday...

About this site

This website is all about learning the hard way. Using a framework like Bootstrap or Foundation could've made development much easier, but where's the fun in that? Working on this site by myself1 has led to some incredible learning experiences, and will hopefully continue to do so in the future. On a similar note, new up-and-coming technologies such as WebAssembly are always welcome here, and will probably be used overzealously. I'll try not to break the site too often, I promise.

In the spirit of learning, I've created a few "guidelines" for this site to abide by, which you can read about on the page linked below. Additionally, all source code relating to this site and its development can be viewed on GitHub.
